# Focus Flow

ADHD-friendly task management app — simple, visual, low-friction.

## Problem

Generic task apps (Todoist, Notion, Things) cause overwhelm for ADHD brains:
- Endless lists
- Too many options
- Analysis paralysis

## Solution

Focus Flow keeps it simple:
- **Today view only** — no infinite backlog
- **3-5 tasks max per day** — forces prioritization
- **Quick capture inbox** — brain dump without judgment
- **Simple streaks** — motivation without gamification overload

## Features

- Daily task limit (configurable 1-10)
- Visual task slots with order numbers
- Inbox for quick captures
- Simple streak tracking
- Celebration animation on completion

## Setup

```bash
cd focus-flow
npm install
# Set up Supabase and add credentials to .env
npm run dev
```

## Tech Stack

- Next.js 14 (App Router)
- Supabase (Auth + Database)
- Tailwind CSS
